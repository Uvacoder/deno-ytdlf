## Usage

**local server**:

`deno run -A --unstable main.ts`

**or using webview**:

`deno run -A --unstable ytfview.ts`

**or from replit instance**: (bugy)

https://ytdlf.sigmasd.repl.co/

## Dependencies

`youtube-dl` needs to be in `PATH`
